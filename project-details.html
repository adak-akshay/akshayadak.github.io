Overview

The Taxi Fleet Management System is designed to efficiently manage taxi operations, including vehicle tracking, booking management, and real-time statistics. It provides a suite of RESTful APIs for managing taxis and bookings, a dashboard for monitoring fleet performance, and JWT-based authentication for secure access.

Key Features:
- Taxi Management: Register, update status, and track the location of taxis.
- Booking Management: Create, update, and manage bookings; assign bookings to nearby taxis.
- Dashboard: Real-time statistics on fleet and booking data.
- Messaging Service: Reliable communication between taxis and the booking system.
- Security: JWT-based authentication for secure system access.

Setup and Running:
- Prerequisites: Java 17, Maven 3.8.5, Docker or Podman.
- Running the Application: Clone the repository, build the Docker image, and run the container. Alternatively, use Podman for a similar setup.

Design Patterns Used:
- Factory Pattern
- Strategy Pattern
- Observer Pattern
- Singleton Pattern

Architecture: Modular design with separate services for taxi management, booking, messaging, and dashboard functionalities.

Conclusion

The Taxi Fleet Management System successfully integrates various components to streamline the operations of a taxi fleet. By leveraging modular architecture and design patterns, it ensures scalability, reliability, and ease of maintenance. The use of JWT for secure authentication and a comprehensive dashboard for real-time statistics enhances both security and usability. This system not only simplifies taxi management and booking processes but also provides valuable insights through its advanced features and real-time data analytics.

The system's focus on robust communication between taxis and the booking platform, along with its ability to handle high volumes of data, makes it a valuable tool for modern taxi fleet operations. Overall, it represents a significant step forward in improving operational efficiency and service quality in the transportation sector.
